# FloraMind

FloraMind é um chatbot de botânica com interface visual e interativa, criado para a disciplina de IA Aplicada ao Desenvolvimento.

## Como rodar

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Insira sua chave da Google AI Studio em backend/.env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Acesse `http://localhost:5173`.
